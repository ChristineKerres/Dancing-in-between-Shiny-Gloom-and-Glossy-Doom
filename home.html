README
The work is an attempt to describe the relationship between users and computerized machines metaphorically. Associatively, both positions are understood as integral components of a performance in which machines are concepts that choreograph their users. For the interaction with computerized machines, the users expose themselves to the framework parameters of the software. The computer program, bound by its conceptual constraints, is defined by the unambiguousness of its code and the design of the user interface. Users in turn, are trapped in a passive role of dependency.
